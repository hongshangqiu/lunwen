\subsection{证据融合器核心代码}

证据融合器（EvidenceBuilder）负责将多源运维数据统一组织为分析上下文，其核心实现如代码所示。

\begin{verbatim}
class EvidenceBuilder:
    def __init__(self, config):
        self.thresholds = config.get('thresholds', {})
        self.severity_rules = config.get('severity_rules', {})

    def build_context(self, question, service, time_range,
                      knowledge, metrics, logs):
        # 提取异常指标
        anomalies = self.extract_anomalies(metrics)
        # 评估严重程度
        severity = self.assess_severity(metrics, logs, anomalies)
        # 构建分析上下文
        context = AnalysisContext(
            question=question,
            service=service,
            time_range=time_range,
            retrieved_knowledge=knowledge,
            metrics_summary=metrics,
            log_summary=logs,
            anomaly_indicators=anomalies,
            severity=severity
        )
        return context
\end{verbatim}

\subsection{RCA引擎核心代码}

RCA引擎（RCAEngine）负责基于大语言模型进行根因分析推理，其核心实现如代码所示。

\begin{verbatim}
class RCAEngine:
    def __init__(self, llm_service, prompts):
        self.llm = llm_service
        self.prompts = prompts

    def analyze(self, context):
        # 构建RCA Prompt
        prompt = self.prompts.build_rca_prompt(context)
        # 调用大语言模型
        response = self.llm.generate(prompt)
        # 解析JSON输出
        result = self.parse_response(response)
        return result
\end{verbatim}
