{\small With the widespread adoption of cloud computing, microservices, and containerization, modern software systems have grown increasingly large and complex, posing significant challenges to traditional operations and maintenance (Ops) workflows. To improve fault diagnosis and operational decision-making in complex systems, this thesis designs and implements an LLM-based intelligent fault diagnosis prototype system for software operations --- LLM-Ops Copilot.}\\[6pt]

{\small First, a domain-specific knowledge base for operations is constructed using the vector database ChromaDB, which supports vectorized storage, semantic retrieval, and metadata filtering by service name and fault type, thereby enhancing the model's ability to acquire task-relevant knowledge. Second, a multi-source data fusion method is proposed. The EvidenceBuilder module integrates log summaries, monitoring metrics, and retrieved knowledge into a unified analysis context, with structured workflows including anomaly extraction and severity assessment. Third, an LLM-based Root Cause Analysis (RCA) engine is designed and implemented. Structured prompts guide the model to generate fault summaries, perform root cause inference, and provide repair suggestions, with confidence scoring and data gap identification. Finally, four comparative experiments are designed to evaluate the effects of knowledge retrieval augmentation, real-time data fusion, and structured prompt design.}\\[6pt]

{\small Experimental results show that relying solely on the LLM for direct responses leads to significant limitations in domain expertise. Introducing RAG-based knowledge augmentation notably improves response relevance and accuracy. Further integrating real-time logs and monitoring metrics enables the system to generate more contextually supported diagnostic results. Structured prompt design effectively enhances output consistency and explainability. Testing across seven typical fault scenarios demonstrates that the proposed method achieves good feasibility in root cause localization accuracy, analysis completeness, and response efficiency, providing a novel technical pathway for intelligent operations and maintenance.}\\[6pt]
